# Meal Prep Tracker

Personal macro tracking and meal planning app. Zero friction, zero backend.

## Stack

Plain HTML + CSS + JS. No build tools. Data in localStorage.

## Deploy (GitHub Pages)

1. Create a new public repo on GitHub named `mealprep`
2. Push this folder:
   ```bash
   git remote add origin https://github.com/YOUR_USERNAME/mealprep.git
   git branch -M main
   git push -u origin main
   ```
3. On GitHub: Settings → Pages → Source: `main` / `/ (root)` → Save
4. Wait ~60 seconds → `https://YOUR_USERNAME.github.io/mealprep/`
